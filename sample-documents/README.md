# Sample documents for the Obsidian bonus task

Use these files as a mixed source folder when practicing document conversion:

- `q3-roadmap-brief.docx` - Word document with headings, bullets, and a table.
- `checkout-customer-research.docx` - Word document with research notes and cross-reference hints.
- `payments-security-review.pdf` - Text-based PDF that should extract cleanly.
- `scanned-vendor-follow-up.pdf` - Image-based PDF for testing Preview Live Text or OCR workflows.

Suggested exercise:

1. Convert the `.docx` files to Markdown with Pandoc.
2. Extract text from the text-based PDF.
3. Try Preview Live Text or OCR for the scanned PDF.
4. Clean the Markdown with Codex.
5. Open the resulting Markdown folder as an Obsidian vault and add cross-links.
